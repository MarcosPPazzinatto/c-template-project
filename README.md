# C Template Project

This repository provides a minimal and clean template for starting new C projects. It includes a basic folder structure, example code, a Makefile, and placeholders for unit testing.

## Features

- Simple and modular directory layout
- Example `main.c` file
- Cross-platform Makefile for compilation
- Support for unit testing (optional)
- License and README included

## Structure

```
c-template-project/
├── src/ # Source files (.c)
├── include/ # Header files (.h)
├── tests/ # Unit tests
├── Makefile # Build instructions
├── README.md # Project documentation
└── LICENSE # License (MIT by default)
```


## Getting Started

1. Clone the repository:
 ```
   git clone https://github.com/your-username/c-template-project.git
 ```
2. Compile the project:

```
  make
```
3. Run the executable:
```
  ./bin/main
```
